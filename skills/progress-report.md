Write a comprehensive client progress report based on the data and context provided below. The report should tell a clear story of the client's journey, celebrate wins, and set a confident direction for what comes next.

## Client Progress Report

### Report Details
- Client name
- Reporting period (e.g., Month 1–3, Program Phase 1 Complete)
- Report date
- Trainer / coach name

### Executive Summary
- 3–5 sentence overview of the client's progress this period
- Overall trajectory: exceeding / meeting / below expectations
- Headline result (the most significant achievement)

### Starting Benchmarks vs. Current Metrics

| Metric | Start | Current | Change |
|---|---|---|---|
| Weight | | | |
| Body fat % | | | |
| Waist | | | |
| Hips | | | |
| Other measurements | | | |
| Key lift 1 (e.g., squat) | | | |
| Key lift 2 (e.g., bench) | | | |
| Cardio benchmark (e.g., 1-mile time) | | | |

### Goal Achievement Review
For each goal set at the start of the period:
- Goal description
- Status: Achieved / Partially achieved / In progress / Not met
- Evidence and context

### Adherence & Consistency
- Training adherence rate (sessions completed / planned)
- Nutrition adherence summary
- Recovery and lifestyle habits
- Barriers encountered and how they were managed

### Strengths & Highlights
- Top 3–5 achievements or improvements to celebrate
- Behavioral or mindset shifts observed
- Skills or confidence gains

### Areas for Development
- Where the client fell short and why (without blame)
- Patterns to address in the next phase
- Specific skills or habits to prioritize

### Trainer's Assessment
- Overall evaluation of the client's effort and progress
- Readiness for the next phase
- Long-term prognosis given current trajectory

### Recommendations & Next Steps

#### Program Adjustments
- Training program changes for the next phase
- Intensity, volume, or modality shifts

#### Nutrition Focus
- Key nutritional adjustments
- New targets if applicable

#### Behavioral Goals
- Habit or lifestyle goals for the next period
- Accountability strategies

#### 90-Day Outlook
- Short-term targets (next 4 weeks)
- Medium-term targets (next 8–12 weeks)
- Key milestones to aim for

### Closing Statement
- Personalized message acknowledging the client's effort
- Motivational framing for the next phase
- Expression of confidence in their continued progress
