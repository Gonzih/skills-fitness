Design a personalized fitness program based on the client details below. Structure the program with clear progressions and all the information needed to execute it.

## Fitness Program Design

### Client Profile Summary
- Name / identifier
- Age, gender, height, weight
- Current fitness level (beginner / intermediate / advanced)
- Health considerations, injuries, or limitations

### Goals & Priorities
- Primary goal (e.g., fat loss, muscle gain, endurance, general fitness)
- Secondary goals
- Target timeline

### Program Overview
- Program name
- Duration (weeks)
- Training frequency (days per week)
- Session length (estimated minutes)
- Training split (e.g., full-body, upper/lower, push/pull/legs)

### Weekly Schedule
- Day-by-day breakdown (training days vs. rest/active recovery)
- Suggested time of day if relevant

### Phase 1 – Foundation (Weeks 1–X)
For each training day:
- Warm-up routine (5–10 min)
- Main workout:
  - Exercise name
  - Sets × Reps (or time/distance)
  - Rest interval
  - Coaching cues or form notes
- Cool-down / mobility work

### Progression Plan
- How and when to increase load, volume, or intensity
- Week-over-week or phase-over-phase progression rules
- Deload protocol (if applicable)

### Cardio & Conditioning
- Type, frequency, duration, and intensity
- Integration with strength sessions

### Recovery Recommendations
- Sleep targets
- Active recovery suggestions
- Mobility or stretching routine

### Equipment Required
- List all equipment needed
- Low-equipment or home alternatives where applicable

### Tracking Metrics
- Key performance indicators to monitor
- How to log progress (reps, load, body measurements, etc.)

## Live Data Sources

- **ExerciseDB API** (exercisedb.io): Fetch exercise library data including muscle groups, equipment requirements, and demonstration images to validate exercise selection and suggest substitutions.
- **Wger Workout Manager API** (wger.de/api/v2): Access nutrition and exercise databases, training logs, and body weight records to enrich program recommendations with real benchmark data.
- **NSCA Training Guidelines**: Reference the National Strength and Conditioning Association's evidence-based guidelines for periodization, load prescription, and exercise technique standards.
- **PubMed Exercise Science Research** (pubmed.ncbi.nlm.nih.gov): Query peer-reviewed exercise science literature to ground program design decisions in current research on training adaptations and programming principles.
